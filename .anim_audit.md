## 4. Fix-Now Punch-List (highest-leverage first)

All fixes are **additive and confined to `Frontend/src/animations.css`** — none touch any R25/R27/R28/R29-frozen component.

1. **One rule kills all four print-safety LOWs.** Add a universal inline-animation disable to the `@media print` block in `animations.css`:
   ```css
   @media print { [style*="animation"] { animation: none !important; } }
   ```
   This covers `.shimmer`, the NursingNotes `pulse` dot, `bedModalIn`, and the 10+ component-scoped keyframes at once — without editing any frozen component. **Highest leverage: 1 line, 4 findings closed.**

2. **Close the reduced-motion `[role=button]` gap** (1 LOW). Append `[role="button"]:not([aria-disabled="true"]):active` to the reduced-motion reset (line 136) so non-native buttons stop receiving `scale(0.97)` under `prefers-reduced-motion`. Mirror the same selector into the `@media print` reset (line 152) to close the matching polish gap in one edit.

3. **Release the GPU layer on `.hga-route`** (1 LOW stacking + 1 polish perf). In the `@media print` block add `.hga-route { will-change: auto; }`, and to fully neutralize the fixed-containing-block edge case, drop the identity transform's persistence (or accept it — impact is a scrolled-tall-page modal pin only). Lowest urgency of the three; do only if touching the file anyway.

4. **Optional one-liner hardening** — cap the stagger so future >12-child containers don't lose their cascade:
   ```css
   .hga-stagger > *:nth-child(n+13) { animation-delay: calc(var(--hga-stagger) * 12); }
   ```

Everything below this line is **defer/won't-fix**: unused `.hga-float`/`.hga-spinner`/`.hga-shimmer` helpers (no consumers, no cost — prune opportunistically or leave), `-webkit-filter` (decade-old browsers only), scrollbar-comment, and the UX cadence/mobile-token taste calls. None block launch.
